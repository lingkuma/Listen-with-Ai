Windows 浏览器音频与录屏测试

这个页面主要验证：麦克风是否可用、录屏时是否能拿到音频轨、最终录制文件里是否包含系统或标签页声音。

预期结果

  • 麦克风：通常可以正常获取。
  • 其他 App / 系统音频：Chromium 浏览器在部分共享模式下可能返回音频轨。
  • 录屏：通常支持，是否带音频取决于浏览器共享弹窗中的选项。

推荐测试步骤

  1. 先点击“检测浏览器能力”。
  2. 点击“申请麦克风”。
  3. 点击“申请录屏 + 音频”，在弹窗中勾选共享音频。
  4. 开始录制，同时播放系统声音或其他窗口声音，录完后回放结果。
