Android 浏览器测试

Android 麦克风 / 设备音频 / 录屏测试

这个页面用于确认 Android 浏览器能否获取麦克风，以及在录屏场景下是否真的返回可用的音频轨。

预期结果

  • 麦克风：大多数现代 Android 浏览器支持。
  • 其他 App 声音：网页通常拿不到，audio: true 往往被忽略。
  • 录屏：有些浏览器支持，但兼容性明显弱于桌面。

推荐测试步骤

  1. 用 HTTPS 环境打开此页。
  2. 先申请麦克风，再尝试申请录屏 + 音频。
  3. 观察日志里 display audio tracks 是否为 0。
  4. 如果能录制，回放结果确认是否仅有麦克风，没有系统音频。
